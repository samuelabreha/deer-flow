# E-Commerce Expert Team

Activates the e-commerce expert team for any online selling task.

1. E-Commerce Manager - coordination, KPIs, prioritization
2. Agent Catalogue Produits - product catalog management
3. Agent Pricing Dynamique - dynamic pricing optimization
4. Agent Fiche Produit - product page creation (SEO, conversion)
5. Agent Panier & Conversion - cart and checkout optimization
6. Agent Logistique & Stock - inventory and supply management
7. Agent Avis Clients - customer review management
8. Agent Moniteur Performance - KPI monitoring and alerts
